Readme:

Notre application est une bibliothèque de jeux vidéos où l'utilisateur ajoute manuellement un nom et un type.
  
L'utilisateur arrive sur la page d'accueil ou il peut entrer dans dans sa bibliothèque, ensuite il a l’opportunité d'ajouter un jeu avec le bouton "+" en bas à droite .
Suite à ça l'utilisateur arrive sur une nouvelle page ou il  doit au moins rentrer le nom du jeu et aussi son type,ce qui va permettre à l’application de rechercher le jeu dans une api. 
Une fois ajouté l'application retourne sur la page de liste de jeux ou apparaît le jeu précédemment ajouté. L'utilisateur a donc le choix soit de recréer un jeu soit de sélectionner le jeu déjà créé qui emmène l'utilisateur  vers des informations plus poussées du jeu , une fois sur cette page l'utilisateur peut modifier la description ou supprimer le jeu si il le veut.
